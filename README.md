# 💰 Tip Calculator

A clean tip and bill splitting calculator.

## Features
- Preset tip buttons: 5%, 10%, 15%, 20%, 25%
- Custom tip percentage input
- Split bill between multiple people
- Shows tip amount, tip per person, and total per person

## Built With
- HTML5
- CSS3
- Vanilla JavaScript

## How to Run
Clone the repo and open `index.html` in your browser.
